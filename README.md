# 🛍️ Ecommerce Website (Frontend)

This is a modern **Ecommerce Web Application Frontend** built using **React, Vite, and Tailwind CSS**.  
The project includes product browsing, filtering, cart functionality, checkout flow, and an admin panel.

---

## 🚀 Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- PayPal Integration

---

## ✨ Features

### 👤 User Features
- Browse products
- View product details
- Filter and sort products
- Add products to cart
- Checkout functionality
- User login & registration
- Order confirmation page
- View order history

---

### 🛒 Cart & Checkout
- Add/remove items from cart
- Cart drawer UI
- Checkout page
- PayPal payment integration
- Order confirmation

---

### 🧑‍💻 Admin Features
- Admin dashboard
- User management
- Product management (UI structure ready)
